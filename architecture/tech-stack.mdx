Stoic Calendar is built with modern React Native technologies, prioritizing performance, type safety, and cross-platform compatibility.

## Core Framework

### React Native

**Version:** 0.81.5

The foundation of the app, providing native iOS and Android rendering from a single JavaScript codebase.

**Key features used:**
- Native animations via Reanimated
- AsyncStorage for persistence
- Platform-specific optimizations
- New Architecture enabled (`newArchEnabled: true`)

### Expo

**SDK Version:** ~54.0.30

Expo provides managed workflow, simplified native module access, and streamlined development.

**Key Expo packages:**

```json
"expo": "~54.0.30",
"expo-router": "~6.0.21",
"expo-constants": "~18.0.12",
"expo-font": "~14.0.10",
"expo-haptics": "~15.0.8",
"expo-image": "~3.0.11",
"expo-splash-screen": "~31.0.13",
"expo-status-bar": "~3.0.9",
"expo-system-ui": "~6.0.9"
```

**Configuration:** See `app.json` for Expo config including:
- Bundle identifiers
- Splash screens
- App icons
- Platform-specific settings

### TypeScript

**Version:** ~5.9.2

Strict TypeScript mode is enabled for maximum type safety.

**Configuration:**
```json
{
  "strict": true,
  "paths": {
    "@/*": ["./*"]
  }
}
```

All components, services, and utilities are fully typed with interfaces defined in `types/`.

## Navigation

### Expo Router

**Version:** ~6.0.21

File-based routing system built on React Navigation.

**Features used:**
- File-based routing in `app/` directory
- Type-safe routes with `typedRoutes: true`
- Deep linking support (scheme: `stoiccalendar://`)
- Automatic navigation types

**Navigation Dependencies:**
```json
"@react-navigation/native": "^7.1.8",
"@react-navigation/bottom-tabs": "^7.4.0",
"@react-navigation/elements": "^2.6.3"
```

## UI & Animations

### React Native Reanimated

**Version:** ~4.1.1

High-performance animations running on the native thread.

**Usage examples:**
- Grid square animations in `components/stoic-grid.tsx`
- Smooth drawer transitions
- Haptic feedback timing

```typescript
import { useSharedValue, withTiming } from 'react-native-reanimated';
```

### React Native Gesture Handler

**Version:** ~2.28.0

Native gesture recognition for smooth interactions.

### Expo Haptics

**Version:** ~15.0.8

Provides tactile feedback for user interactions:

```typescript
import * as Haptics from 'expo-haptics';

await Haptics.impactAsync(Haptics.ImpactFeedbackStyle.Light);
```

### Icon Libraries

**Phosphor React Native**
- **Version:** ^3.0.2
- Beautiful, consistent icon set
- Used throughout the app for UI elements

**Expo Vector Icons**
- **Version:** ^15.0.3
- Fallback icon library

### Fonts

**Cormorant Garamond**
- **Version:** ^0.4.1
- Elegant serif font for Stoic aesthetic
- Loaded via `@expo-google-fonts/cormorant-garamond`

## Data & Storage

### AsyncStorage

**Package:** `@react-native-async-storage/async-storage`
**Version:** ^2.2.0

Local key-value storage for:
- Timeline data
- User settings
- Active timeline state

Wrapped in `services/storage.ts` for centralized access.

### Date/Time Picker

**Package:** `@react-native-community/datetimepicker`
**Version:** ^8.5.1

Native date pickers for timeline creation:
- iOS native picker
- Android Material Design picker

## Monetization

### RevenueCat

**Packages:**
```json
"react-native-purchases": "^9.6.13",
"react-native-purchases-ui": "^9.6.13"
```

In-app subscription management:
- Paywall UI in `app/paywall.tsx`

**Features:**
- Cross-platform subscription handling
- Receipt validation
- Subscription status tracking
- Restore purchases

## iOS Widgets

### App Groups Module

**Package:** `app-groups` (local module in `modules/app-groups`)

Custom native module for sharing data between the main app and iOS widgets.

**Usage:**
```typescript
import AppGroups from 'app-groups';

await AppGroups.setItem('group.com.stoiccalendar.shared', 'activeTimeline', data);
```

### Widget Implementation

Swift-based widget in `targets/StoicGridWidget/` using:
- WidgetKit
- SwiftUI
- Timeline providers

### iOS
