## Overview

Stoic Calendar uses a comprehensive theme system based on Apple's Human Interface Guidelines. The theme is defined in `constants/theme.ts` and provides a complete design token system for colors, typography, spacing, and more.

## Design Philosophy

**NorthStar Principle**: "Make time clearer without making it louder"

**Key Principles**:
- Dark-mode-first design
- Minimal, calm, neutral aesthetic
- Pure black backgrounds for OLED displays
- Subtle animations (no spring/bounce)
- iOS 26 design patterns

## Theme Modes

The app supports three theme modes:

1. **Light** - Light mode colors
2. **Dark** - Dark mode colors (default)
3. **System** - Follow system appearance

### Theme Mode Selection

Theme preference is stored in AsyncStorage and loaded in the root layout:

```tsx
import { getThemeMode, setThemeMode } from '@/services/storage';
import { ThemeMode } from '@/types/timeline';

// Get current theme
const themeMode: ThemeMode = await getThemeMode();

// Set theme
await setThemeMode('dark');
await setThemeMode('light');
await setThemeMode('system');
```

### System Appearance

Use React Native's `useColorScheme()` hook:

```tsx
import { useColorScheme, Appearance } from 'react-native';

const colorScheme = useColorScheme(); // 'light' | 'dark' | null

// Force appearance
Appearance.setColorScheme('dark');
Appearance.setColorScheme(null); // System default
```

## Best Practices

### 1. Always Use Theme Constants

```tsx
// Good
import { Spacing, Colors } from '@/constants/theme';
const colors = Colors[colorScheme ?? 'dark'];

// Bad
const padding = 16; // Use Spacing.md instead
const textColor = '#FFFFFF'; // Use colors.textPrimary
```

### 2. Respect Dark Mode First

Default to dark mode when color scheme is unknown:

```tsx
const colors = Colors[colorScheme ?? 'dark'];
```

### 3. Use Semantic Color Names

```tsx
// Good
backgroundColor: colors.cardBackground,
color: colors.textPrimary,

// Bad
backgroundColor: '#1C1C1E',
color: '#FFFFFF',
```

### 4. Follow Spacing Scale

```tsx
// Good
padding: Spacing.md,
marginBottom: Spacing.lg,

// Bad
padding: 15, // Not in scale
marginBottom: 20, // Use Spacing constants
```

### 5. Use Spread for Shadows

```tsx
const styles = StyleSheet.create({
  card: {
    ...Shadows.medium,
    backgroundColor: colors.cardBackground,
  },
});
```
