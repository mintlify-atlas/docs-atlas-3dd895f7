## Planned Architecture

### Data Structure

Timelines will be stored in Firestore with this structure:

```
users/
  {userId}/
    timelines/
      {timelineId}/
        - id: string
        - type: string
        - title: string
        - startDate: string (ISO 8601)
        - endDate: string (ISO 8601)
        - createdAt: string (ISO 8601)
        - isActive: boolean
        - widgetPreferences: object (optional)
```

**Why it's Firebase-ready**:
- All dates use ISO 8601 strings (Firestore Timestamp compatible)
- Flat structure (no deeply nested objects)
- Unique IDs (UUID v4)
- No circular references

### Authentication Providers

Planned authentication methods:

1. **Google Sign-In**
2. **Apple Sign-In** (required for iOS App Store)
3. Anonymous auth (optional for free tier)

### syncTimelines()

Sync local AsyncStorage with Firestore:

```typescript
export async function syncTimelines(): Promise<void>
```

**Implementation plan**:

1. Get current user ID from auth
2. Fetch timelines from Firestore for this user
3. Merge with local timelines (conflict resolution)
4. Update both local AsyncStorage and Firestore
5. Set up real-time listeners for changes

**Conflict resolution strategy**:
- Compare `createdAt` and `updatedAt` timestamps
- Newer version wins
- Deleted items tracked separately

**Example implementation**:

```typescript
import { collection, query, getDocs, onSnapshot } from 'firebase/firestore';
import { loadTimelines, saveTimelines } from '@/services/storage';

export async function syncTimelines(): Promise<void> {
  const userId = auth.currentUser?.uid;
  if (!userId) throw new Error('Not authenticated');

  // Fetch remote timelines
  const q = query(collection(db, `users/${userId}/timelines`));
  const snapshot = await getDocs(q);
  const remoteTimelines = snapshot.docs.map(doc => doc.data() as Timeline);

  // Fetch local timelines
  const localTimelines = await loadTimelines();

  // Merge (newer wins)
  const merged = mergeTimelines(localTimelines, remoteTimelines);

  // Update both
  await saveTimelines(merged);
  await uploadAllTimelines(merged);

  // Listen for changes
  onSnapshot(q, (snapshot) => {
    // Update local when remote changes
  });
}
```

```typescript
import * as AppleAuthentication from 'expo-apple-authentication';
import { signInWithCredential, OAuthProvider } from 'firebase/auth';

export async function signInWithApple(): Promise<void> {

  const credential = await AppleAuthentication.signInAsync({
```

## Real-Time Sync

Listen for changes across devices:

```typescript
import { onSnapshot, collection, query } from 'firebase/firestore';

function listenToTimelines(userId: string, callback: (timelines: Timeline[]) => void) {
  const q = query(collection(db, `users/${userId}/timelines`));
  
  return onSnapshot(q, (snapshot) => {
    const timelines = snapshot.docs.map(doc => doc.data() as Timeline);
    callback(timelines);
  });
}

// Usage
const unsubscribe = listenToTimelines(userId, async (timelines) => {
  // Update local storage
  await saveTimelines(timelines);
  
  // Update UI
  setTimelines(timelines);
});

// Cleanup
unsubscribe();
```
