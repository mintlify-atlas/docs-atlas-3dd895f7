## Overview

The Timeline Calculator Service (`services/timeline-calculator.ts`) provides all business logic for creating timelines, calculating statistics, and managing timeline updates. It handles the four timeline types (YEAR, MONTH, WEEK, CUSTOM) and auto-update logic for dynamic timelines.

## Timeline Creation

### Calculate Timeline from Type

Generate timeline configuration from a timeline type:

```typescript
export function calculateTimelineFromType(
  type: TimelineType,
  customOptions?: {
    year?: number;
    month?: number;
    startDate?: Date;
    endDate?: Date;
    title?: string;
  }
): TimelineConfig
```

**Parameters:**
- `type`: Timeline type (YEAR, MONTH, WEEK, CUSTOM)
- `customOptions`: Optional configuration based on type

**Returns:** Timeline configuration with start/end dates and title

**Examples:**

```typescript
import { calculateTimelineFromType, TimelineType } from '@/services/timeline-calculator';

// Current year
const yearConfig = calculateTimelineFromType(TimelineType.YEAR);
// { startDate: "2026-01-01T00:00:00.000Z", endDate: "2026-12-31T23:59:59.999Z", title: "2026" }

// Specific year
const year2027 = calculateTimelineFromType(TimelineType.YEAR, { year: 2027 });
// { startDate: "2027-01-01T00:00:00.000Z", endDate: "2027-12-31T23:59:59.999Z", title: "2027" }

// Current month
const monthConfig = calculateTimelineFromType(TimelineType.MONTH);
// { startDate: "2026-03-01T00:00:00.000Z", endDate: "2026-03-31T23:59:59.999Z", title: "March" }

// Current week (Monday-Sunday)
const weekConfig = calculateTimelineFromType(TimelineType.WEEK);
// { startDate: "2026-03-02T00:00:00.000Z", endDate: "2026-03-08T23:59:59.999Z", title: "This Week" }

// Custom date range
const customConfig = calculateTimelineFromType(TimelineType.CUSTOM, {
  startDate: new Date('2026-06-01'),
  endDate: new Date('2026-08-31'),
  title: 'Summer 2026',
});
// { startDate: "2026-06-01T00:00:00.000Z", endDate: "2026-08-31T23:59:59.999Z", title: "Summer 2026" }
```

### Create Timeline

```typescript
export function createTimeline(
  type: TimelineType,
  customOptions?: {
    year?: number;
    month?: number;
    startDate?: Date;
    endDate?: Date;
    title?: string;
    isActive?: boolean;
  }

```

**Parameters:**
- `type`: Timeline type
- `customOptions`: Optional configuration and `isActive` flag

**Returns:** Complete Timeline object with unique ID and creation timestamp

**ID Format:** `timeline_{timestamp}_{random}` (e.g., `timeline_1709740800000_abc123xyz`)

**Example:**
```typescript

import { TimelineType } from '@/types/timeline';
import { saveTimeline } from '@/services/storage';

const newTimeline = createTimeline(TimelineType.YEAR, {
  year: 2026,
  isActive: true,
});

await saveTimeline(newTimeline);
```

## Timeline Updates

### Update Week Timeline

Update a "This Week" timeline to the current week:

```typescript
export function updateWeekTimeline(timeline: Timeline): Timeline
```

**Behavior:**
- Throws error if timeline type is not WEEK
- Returns new timeline with updated start/end dates
- Preserves all other properties (id, createdAt, isActive, etc.)

**Example:**
```typescript
import { updateWeekTimeline, weekTimelineNeedsUpdate } from '@/services/timeline-calculator';
import { getActiveTimeline, saveTimeline } from '@/services/storage';

const timeline = await getActiveTimeline();
if (timeline && timeline.type === 'week' && weekTimelineNeedsUpdate(timeline)) {
  const updated = updateWeekTimeline(timeline);
  await saveTimeline(updated);
}
```

### Auto-Update Any Timeline

Automatically detect and update any timeline that needs updating:

```typescript
export function updateTimelineIfNeeded(
  timeline: Timeline
): { timeline: Timeline; wasUpdated: boolean }
```

**Returns:** Object with updated timeline and boolean flag

**Behavior:**
- Automatically detects timeline type
- Checks if update is needed
- Returns updated timeline if needed, otherwise returns original
- CUSTOM timelines never auto-update

**Example:**
```typescript
import { updateTimelineIfNeeded } from '@/services/timeline-calculator';
import { loadTimelines, saveTimeline } from '@/services/storage';

const timelines = await loadTimelines();
for (const timeline of timelines) {
  const { timeline: updated, wasUpdated } = updateTimelineIfNeeded(timeline);
  if (wasUpdated) {
    console.log(`Updated timeline: ${updated.title}`);
    await saveTimeline(updated);
  }
}
```

## Timeline Validation

### Validate Timeline

Validate a timeline object for required fields and valid dates:

```typescript
export function validateTimeline(timeline: Partial<Timeline>): boolean
```

**Validation checks:**
- Has valid ID
- Has valid type from TimelineType enum
- Has title
- Has start and end dates
- Dates are valid ISO 8601 strings
- Start date is before end date
- Has creation date

**Throws:** Error with descriptive message if validation fails

**Example:**
```typescript
import { validateTimeline } from '@/services/timeline-calculator';

try {
  validateTimeline(timeline);
  console.log('Timeline is valid');
} catch (error) {
  console.error('Validation failed:', error.message);
}
```

## Usage Patterns

### Creating and Saving a Timeline

```typescript
import { createTimeline } from '@/services/timeline-calculator';
import { saveTimeline, setActiveTimeline } from '@/services/storage';
import { TimelineType } from '@/types/timeline';

// Create a year timeline
const timeline = createTimeline(TimelineType.YEAR, { year: 2026 });

// Save to storage
await saveTimeline(timeline);

// Set as active
await setActiveTimeline(timeline.id);
```

### Displaying Timeline Progress

```typescript
import { calculateTimelineStats, getTimelineProgress } from '@/services/timeline-calculator';
import { getActiveTimeline } from '@/services/storage';

const timeline = await getActiveTimeline();
if (timeline) {
  const stats = calculateTimelineStats(timeline);
  const progressText = getTimelineProgress(timeline);
  
  console.log(progressText); // "65 of 365 days"
  console.log(`${stats.progressPercentage}% complete`);
}
```

### Auto-Updating Week Timeline on App Launch

```typescript
import { updateTimelineIfNeeded } from '@/services/timeline-calculator';
import { getActiveTimeline, saveTimeline } from '@/services/storage';

const timeline = await getActiveTimeline();
if (timeline) {
  const { timeline: updated, wasUpdated } = updateTimelineIfNeeded(timeline);
  if (wasUpdated) {
    await saveTimeline(updated);
    console.log('Active timeline updated to current period');
  }
}
```
