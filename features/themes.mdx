#### Dark Mode Colors

- **Background**: `#000000` (Pure black)
- **Secondary Background**: `#1C1C1E` (Dark gray)
- **Tertiary Background**: `#2C2C2E` (Slightly lighter gray)
- **Primary Text**: `#FFFFFF` (White)

- **Background**: `#FFFFFF` (Pure white)

- **Primary Text**: `#000000` (Black)

- **Accent**: `#007AFF` (iOS blue)

<Note>
  Both light and dark modes use the same iOS system blue (`#007AFF`) for accents, ensuring consistency across your device.
</Note>

## Grid Color Themes

Grid color themes change the appearance of filled days in your timeline grid. All themes use neutral gray for empty (future) days to maintain high contrast.

### Available Themes

<Tabs>
  <Tab title="Classic Blue">
    **The default iOS look**
    
    - **Filled Days (Light)**: `#007AFF` (iOS Blue)
    - **Filled Days (Dark)**: `#007AFF` (iOS Blue)
    - **Empty Days**: Neutral gray
    
    Classic Blue uses the standard iOS system blue, providing a familiar and clean appearance that matches the iOS ecosystem.
  </Tab>
  
  <Tab title="Forest Green">
    **Natural and calming**
    
    - **Filled Days (Light)**: `#34C759` (System Green)
    - **Filled Days (Dark)**: `#30D158` (Bright Green)
    - **Empty Days**: Neutral gray
    
    Forest Green offers a nature-inspired palette perfect for habit tracking and wellness goals.
  </Tab>
  
  <Tab title="Sunset Orange">
    **Warm and energetic**
    
    - **Filled Days (Light)**: `#FF9500` (System Orange)
    - **Filled Days (Dark)**: `#FF9F0A` (Bright Orange)
    - **Empty Days**: Neutral gray
    
    Sunset Orange provides warmth and energy, ideal for creative projects and motivation.
  </Tab>
  
  <Tab title="Monochrome">
    **Pure simplicity**
    
    - **Filled Days (Light)**: `#000000` (Black)
    - **Filled Days (Dark)**: `#FFFFFF` (White)
    - **Empty Days**: Neutral gray
    
    Monochrome delivers the most minimal aesthetic, removing all color distraction for pure focus on time passage.
  </Tab>
</Tabs>

### Changing Grid Color Theme

<Steps>
  <Step title="Open Settings">
    Tap the Settings tab at the bottom of the app
  </Step>
  <Step title="Find Appearance">
    Scroll to the Appearance section
  </Step>
  <Step title="Tap Grid Color Theme">

<Tip>
  Grid color themes apply to both the main app and widgets. Change your theme to match your home screen wallpaper or your current mood.

## Design Philosophy

Stoic Calendar follows a **dark-mode-first** design philosophy inspired by Stoic principles of clarity and minimalism.

### Design Principles

<AccordionGroup>
  <Accordion title="Make time clearer without making it louder">
    Visual elements are intentionally subtle. The grid speaks for itself without flashy animations or bright colors.
  </Accordion>
  
  <Accordion title="Dark mode as the default">
    Pure black (`#000000`) backgrounds reduce eye strain and emphasize the content. OLED displays benefit from true black pixels.
  </Accordion>
  
  <Accordion title="Minimal color palette">
    Limited color choices prevent decision fatigue. Four carefully curated grid themes cover all aesthetic preferences.
  </Accordion>
  
  <Accordion title="iOS-native appearance">
    Uses Apple's system colors and follows Human Interface Guidelines for a familiar, comfortable experience.
  </Accordion>
  
  <Accordion title="No gamification">
    No badges, streaks, or confetti. The grid visualization alone provides awareness without external motivation.
  </Accordion>
</AccordionGroup>

## Animations

Animations in Stoic Calendar are subtle and non-distracting.

### Animation Principles

- **Fast**: 200ms for quick interactions (button taps)
- **Normal**: 300ms for standard transitions (screen changes)
- **Slow**: 500ms for deliberate actions (modal presentations)

### Easing

All animations use `ease-in-out` easing. No spring or bounce effects—keeping the experience calm and predictable.

<Warning>
  Stoic Calendar intentionally avoids flashy animations. The goal is awareness, not entertainment.
</Warning>

## Technical Reference

### Theme Constants

All theme values are defined in `constants/theme.ts:16`:

- Color system: `Colors.light` and `Colors.dark`
- Grid palettes: `GridColorPalettes`
- Typography: `Fonts`, `FontSizes`, `FontWeights`
- Spacing: `Spacing`, `BorderRadius`
- Animations: `Animations`, `Haptics`

### Theme Context

The app uses React Context to provide theme values throughout the component tree. Theme changes trigger immediate re-renders.

### Widget Theme Sync

Theme settings automatically sync to widgets via App Groups. See `services/widget-data-service.ts:229` for implementation details.
