### Managing Your Subscription

All subscriptions are managed through the App Store.

#### Cancel Subscription

1. Open **Settings** on your iPhone
2. Tap your **name** at the top
3. Tap **Subscriptions**
4. Select **Stoic Calendar Pro**
5. Tap **Cancel Subscription**

<Warning>
  If you cancel, you'll keep Pro access until the end of your current billing period. Your subscription won't auto-renew.
</Warning>

#### Change Subscription Plan

1. Open **Settings** on your iPhone
2. Tap your **name** at the top
3. Tap **Subscriptions**
4. Select **Stoic Calendar Pro**
5. Choose a different plan (Monthly or Yearly)

#### View Billing History

1. Open **Settings** on your iPhone
2. Tap your **name** at the top
3. Tap **Media & Purchases**
4. Tap **Purchase History**

## Technical Details

### RevenueCat Integration

Stoic Calendar uses [RevenueCat](https://www.revenuecat.com/) for subscription management.

**Benefits:**
- Cross-platform subscription management
- Automatic receipt validation
- Real-time entitlement updates
- Server-side subscription status

### Entitlements

Pro features are controlled by the **"Memento Calendar Pro"** entitlement.

```typescript
const hasProAccess = await isPro();
```

The app checks this entitlement to determine feature access:

- `true`: User has active Pro subscription
- `false`: User is on free tier

### Product IDs

- **Monthly**: `memento_monthly`
- **Yearly**: `memento_yearly`

Both products grant the same "Memento Calendar Pro" entitlement.

### Anonymous User IDs

RevenueCat generates an anonymous user ID for each installation. You can find this in Settings → About for customer support purposes.

<Note>
  Your anonymous user ID is **not** linked to your Apple ID. It's only used for subscription management.
</Note>

## Free Tier Limits

The free version includes limits to encourage upgrading to Pro:

  <Card title="Timeline Limit" icon="list">
    **Maximum**: 3 timelines

  </Card>

    **Maximum**: 1 custom timeline
    
    Additional custom timelines require Pro
  </Card>
  <Card title="Widget Size Limit" icon="square">
    **Blocked**: Large home screen widget
    
    Small and medium widgets remain available
  </Card>
  <Card title="No Cloud Sync" icon="cloud-slash">
    **Future Feature**: Cloud sync is Pro-only
    
    Free users store data locally
  </Card>
</CardGroup>

### What Happens When You Hit Limits?

**Timeline Limit (3 timelines)**:
- You'll see a prompt to upgrade when trying to create a 4th timeline
- You can delete an existing timeline to make room
- Or upgrade to Pro for unlimited timelines

**Custom Timeline Limit (1 custom)**:
- You'll see a Pro upgrade prompt when creating a 2nd custom timeline
- You can delete your existing custom timeline to create a new one
- Year, Month, and Week timelines don't count toward the custom limit

**Large Widget**:
- Large widget option shows a "Pro Only" badge
- Tapping it opens the upgrade screen
- You can still use small and medium widgets

### Purchasing Pro

```typescript

const offerings = await getOfferings();
if (offerings?.monthly) {
  const { customerInfo, userCancelled } = await purchasePackage(offerings.monthly);
  if (!userCancelled) {
    // Purchase successful
  }
}
```
