Pure functions for date calculations in Stoic Calendar. All dates use ISO 8601 format strings for consistency and Firebase compatibility. Week starts on Monday (ISO 8601 standard).

## Core Date Functions

### getStartOfToday

Get the start of today (00:00:00).

```typescript
function getStartOfToday(): Date
```

<ResponseField name="return" type="Date">
  Date object set to today at 00:00:00
</ResponseField>

**Example:**

```typescript
import { getStartOfToday } from '@/utils/date-helpers';

const todayStart = getStartOfToday();
console.log(todayStart); // 2026-03-03T00:00:00.000Z
```

## Timeline Calculation Functions

### getDaysBetween

Calculate the number of days between two dates (inclusive).

```typescript
function getDaysBetween(startDate: Date | string, endDate: Date | string): number
```

<ParamField path="startDate" type="Date | string" required>
  Start date (Date object or ISO 8601 string)
</ParamField>

<ParamField path="endDate" type="Date | string" required>
  End date (Date object or ISO 8601 string)
</ParamField>

<ResponseField name="return" type="number">
  Number of days between dates (inclusive). Example: Jan 1 to Jan 3 = 3 days
</ResponseField>

**Example:**

```typescript
import { getDaysBetween } from '@/utils/date-helpers';

const days = getDaysBetween('2026-01-01', '2026-01-03');
console.log(days); // 3 (Jan 1, Jan 2, Jan 3)
```

### getDaysPassed

Calculate days passed in a timeline (from startDate to today). Returns 0 if timeline hasn't started yet. Today is counted as "passed" once it starts (at 00:00).

```typescript
function getDaysPassed(startDate: Date | string, endDate: Date | string): number
```

<ParamField path="startDate" type="Date | string" required>
  Timeline start date
</ParamField>

<ParamField path="endDate" type="Date | string" required>
  Timeline end date
</ParamField>

<ResponseField name="return" type="number">
  Number of days passed from start to today (0 if timeline hasn't started)
</ResponseField>

**Example:**

```typescript
import { getDaysPassed } from '@/utils/date-helpers';

// Today is 2026-03-03
const passed = getDaysPassed('2026-01-01', '2026-12-31');
console.log(passed); // 62 (days from Jan 1 to today)
```

### getDaysRemaining

Calculate days remaining in a timeline (from tomorrow to endDate). Returns 0 if timeline has ended. Today is NOT counted in "remaining" since it's already in "passed".

```typescript
function getDaysRemaining(startDate: Date | string, endDate: Date | string): number
```

<ParamField path="startDate" type="Date | string" required>
  Timeline start date
</ParamField>

<ParamField path="endDate" type="Date | string" required>
  Timeline end date
</ParamField>

<ResponseField name="return" type="number">
  Number of days remaining from tomorrow to end (0 if timeline has ended)
</ResponseField>

**Example:**

```typescript
import { getDaysRemaining } from '@/utils/date-helpers';

// Today is 2026-03-03
const remaining = getDaysRemaining('2026-01-01', '2026-12-31');
console.log(remaining); // 303 (days from tomorrow to Dec 31)
```

## Date Manipulation Functions

### addDays

Add days to a date. Returns a new Date object.

```typescript
function addDays(date: Date | string, days: number): Date
```

<ParamField path="date" type="Date | string" required>
  Starting date
</ParamField>

<ParamField path="days" type="number" required>
  Number of days to add (can be negative)
</ParamField>

<ResponseField name="return" type="Date">
  New Date object with days added
</ResponseField>

**Example:**

```typescript
import { addDays } from '@/utils/date-helpers';

const start = new Date('2026-01-01');
const future = addDays(start, 10);
console.log(future); // 2026-01-11

const past = addDays(start, -5);
console.log(past); // 2025-12-27
```

### getDateFromDotIndex

Get date from timeline start and dot index. Dot index is 0-based (0 = first day of timeline).

```typescript
function getDateFromDotIndex(startDate: Date | string, dotIndex: number): Date
```

<ParamField path="startDate" type="Date | string" required>
  Timeline start date
</ParamField>

<ParamField path="dotIndex" type="number" required>
  0-based index of the dot in the grid
</ParamField>

<ResponseField name="return" type="Date">
  Date corresponding to the dot index
</ResponseField>

**Example:**

```typescript
import { getDateFromDotIndex } from '@/utils/date-helpers';

// For a year timeline starting Jan 1, 2026
const date0 = getDateFromDotIndex('2026-01-01', 0);
console.log(date0); // 2026-01-01 (first day)

const date10 = getDateFromDotIndex('2026-01-01', 10);
console.log(date10); // 2026-01-11 (11th day)
```
